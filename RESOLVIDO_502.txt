O código fonte está em outro lugar ou foi deployado
de outra forma. Para fazer deploy novo:

1. Conectar no servidor
2. Ver onde está o código fonte original
3. Fazer git pull/atualização
4. Executar: python manage.py migrate
5. Executar: python manage.py collectstatic
6. Reiniciar serviços
