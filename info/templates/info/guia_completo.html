Introdução ao Sistema

O Sistema de Agendamentos é uma solução completa e moderna para gerenciar agendamentos, clientes e serviços de forma eficiente, organizada e profissional.

🚀 Principais Funcionalidades:

  • Gestão completa de agendamentos
  • Cadastro e controle de clientes
  • Catálogo de serviços personalizável
  • Sistema de autenticação seguro
  • Relatórios e análises avançadas
  • Dashboard interativo e intuitivo
  • Controle de status em tempo real
  • Interface responsiva e moderna
Como usar este guia: Este documento está organizado por módulos funcionais. Use o índice lateral para navegar rapidamente entre as seções ou leia sequencialmente para um aprendizado completo.

Sistema de Autenticação

O sistema possui um robusto módulo de autenticação que garante a segurança dos dados e controla o acesso às funcionalidades.

🔐 Funcionalidades de Segurança:

  • Login seguro com validação de credenciais
  • Controle de sessões ativas
  • Logout automático por inatividade
  • Proteção contra ataques CSRF
  • Criptografia de senhas
📝 Como fazer Login:
  1. Acesse a página inicial do sistema
  2. Clique em "Entrar" ou acesse /login/
  3. Digite seu nome de usuário
  4. Digite sua senha
  5. Clique em "Entrar"
  6. Você será redirecionado para o dashboard
🚪 Como fazer Logout:
  1. Clique no seu nome no menu superior
  2. Selecione "Sair" no menu dropdown
  3. Confirme a ação se solicitado
  4. Você será desconectado automaticamente
  5. Será redirecionado para a página de login
Importante para sua Segurança: Sempre faça logout ao terminar de usar o sistema, especialmente em computadores compartilhados ou públicos. Nunca compartilhe suas credenciais de acesso com terceiros.

Gestão de Agendamentos

O módulo de agendamentos é o coração do sistema, permitindo criar, editar, acompanhar e gerenciar todos os compromissos de forma organizada e eficiente.

📊 Status dos Agendamentos:

Agendado Agendamento criado, aguardando confirmação do cliente
Confirmado Cliente confirmou presença no horário marcado
Em Andamento Serviço está sendo executado no momento
Concluído Serviço finalizado com sucesso
Cancelado Agendamento cancelado por algum motivo
Não Compareceu Cliente não compareceu no horário agendado

➕ Criando um Novo Agendamento:

  1. Acesse o formulário: Dashboard → "Novo Agendamento" ou Menu → "Agendamentos" → "Criar"
  2. Selecione o Cliente: Escolha um cliente existente na lista ou clique em "Novo Cliente" para cadastrar
  3. Escolha o Serviço: Selecione o serviço que será realizado no dropdown
  4. Defina Data e Hora: Use o calendário para escolher data e o campo de hora para o horário
  5. Adicione Observações: Inclua informações importantes ou especiais (campo opcional)
  6. Revise os Dados: Confira todas as informações antes de salvar
  7. Salve o Agendamento: Clique em "Salvar" para criar o agendamento

✏️ Editando um Agendamento:

  1. Acesse a lista de agendamentos
  2. Clique no ícone de edição (lápis) do agendamento desejado
  3. Modifique as informações necessárias
  4. Clique em "Salvar Alterações"
Dica Inteligente: O sistema verifica automaticamente conflitos de horário e não permite agendamentos sobrepostos. Além disso, calcula automaticamente o horário de término baseado na duração do serviço selecionado.

Gestão de Clientes

O módulo de clientes permite manter um cadastro completo e organizado de todos os seus clientes, com histórico de agendamentos e informações de contato.

👤 Cadastrando um Novo Cliente:

  1. Acesse "Clientes" no menu principal
  2. Clique no botão "Novo Cliente"
  3. Preencha as informações obrigatórias:
    • Nome completo: Nome e sobrenome do cliente
    • Telefone: Número para contato (com DDD)
    • Email: Endereço de email (opcional mas recomendado)
  4. Adicione informações complementares:
    • Endereço completo
    • Data de nascimento
    • Observações especiais
  5. Clique em "Salvar Cliente"

🔍 Gerenciando e Organizando Clientes:

Busca e Filtros Avançados
  • Busque por nome completo ou parcial
  • Pesquise por número de telefone
  • Filtre por email cadastrado
  • Use filtros de status (ativo/inativo)
  • Visualize histórico completo de agendamentos
Ações Disponíveis
  • Editar informações pessoais
  • Ver detalhes e histórico completo
  • Criar novo agendamento diretamente
  • Marcar como cliente VIP
  • Excluir cliente (com confirmação)
Organização Inteligente: O sistema mantém automaticamente estatísticas de cada cliente, como número total de agendamentos, data do último atendimento e frequência de visitas, ajudando você a identificar clientes VIP.

Catálogo de Serviços

O módulo de serviços permite criar e gerenciar um catálogo completo de todos os serviços oferecidos, com preços, durações e descrições detalhadas.

⚙️ Cadastrando um Novo Serviço:

  1. Acesse "Serviços" no menu principal
  2. Clique em "Novo Serviço"
  3. Preencha as informações essenciais:
    • Nome do Serviço: Título claro e descritivo
    • Descrição: Detalhes do que está incluído
    • Preço: Valor cobrado pelo serviço
    • Duração: Tempo estimado (horas e minutos)
  4. Configure o status (Ativo/Inativo)
  5. Clique em "Salvar Serviço"

💡 Dicas para Cadastro de Serviços:

Nomenclatura
  • Use nomes claros e específicos
  • Evite abreviações confusas
  • Seja consistente na nomenclatura
  • Exemplo: "Corte Masculino Tradicional"
Precificação
  • Pesquise preços da concorrência
  • Considere custos operacionais
  • Revise preços periodicamente
  • Ofereça pacotes promocionais
Duração Inteligente: A duração do serviço é usada pelo sistema para calcular automaticamente o horário de término dos agendamentos e verificar disponibilidade de horários, evitando conflitos.

Relatórios e Analytics

O sistema oferece uma suite completa de relatórios e análises para acompanhar o desempenho do seu negócio e tomar decisões baseadas em dados.

📊 Tipos de Relatórios Disponíveis:

Dashboard Executivo
  • KPIs principais em tempo real
  • Gráfico de agendamentos por dia
  • Próximos agendamentos
  • Resumo mensal automático
  • Indicadores de crescimento
Relatórios Detalhados
  • Faturamento detalhado por período
  • Ranking de serviços mais procurados
  • Top clientes mais frequentes
  • Análise de crescimento mensal
  • Projeções e tendências

🔧 Como Usar os Filtros de Relatórios:

  1. Acesse a página "Relatórios" no menu
  2. Clique no botão "Filtros" no canto superior
  3. Selecione o período desejado usando os campos de data
  4. Use os botões de período rápido (7, 30, 90 dias, 1 ano)
  5. Clique em "Aplicar Filtros" para atualizar os dados
  6. Use "Limpar Filtros" para voltar à visualização padrão

📈 Interpretando os KPIs:

Faturamento Receita total do período selecionado
Ticket Médio Valor médio por atendimento realizado
Total Atendimentos Número de agendamentos concluídos
Crescimento Percentual de crescimento em relação ao período anterior
Dica de Gestão: Monitore os relatórios semanalmente para identificar tendências, padrões de comportamento dos clientes e oportunidades de crescimento. Use os dados para ajustar preços, horários e estratégias de marketing.
