=================================================
COMANDOS PARA RESOLVER 502
=================================================

# 1. Ir para o diretório do projeto

# 2. Reiniciar o Django/Gunicorn
sudo supervisorctl restart s-agendamento

# 3. Recarregar o Nginx
sudo systemctl reload nginx

# 4. Verificar se está rodando
sudo supervisorctl status s-agendamento

# 5. Sair (opcional)
exit

=================================================

Copie e cole tudo isso no terminal onde você está:

cd /opt/s-agendamento && sudo supervisorctl restart s-agendamento && sudo systemctl reload nginx && sudo supervisorctl status

=================================================
SE QUISER FAZER DEPLOY COMPLETO:
=================================================

cd /opt/s-agendamento

git reset --hard origin/main
source venv/bin/activate
pip install -r requirements.txt --upgrade --quiet
python manage.py migrate
python manage.py collectstatic --noinput
sudo supervisorctl restart s-agendamento
sudo systemctl reload nginx
sudo supervisorctl status
